Design Statement

Hollow Knight is a game I recently got into not only thanks to the sequel 'Silksong' coming out this year, but the discount on the original game itself, and encouragement from my friends. The style of the game is really nice and I was lucky that a lot of the images I put in have a black foreground that created a vignette around each picture. This allowed me to make each page have a black background that would make the images look like they're blended nicely together. Considering Hollow Knight is a fantasy type game I also wanted to incorporate fonts that would represent that. For the buttons I made sure that they fit a similar color to what Hollow Knight's general color pallet has besides black and white. There are lots of blues and purples for example. It's simplistic but visually appealing. I wanted all the text to be in the middle and below each image, acting as a description and to guide the audience's eyes easily as well.
